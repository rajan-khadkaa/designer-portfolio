# Rajan - Designer Portfolio

A personal portfolio showcasing UI/UX case studies, design process, and skills.

Live demo: [https://www.rajan-khadka.com.np/](https://www.rajan-khadka.com.np/)

## Overview

This portfolio presents end-to-end product design work, covering research, wireframes, testing, and final high-fidelity designs. It includes detailed case studies along with an overview of my design process, skills, and tools.

## Case Studies

- **HRMS** — A web-based HR management system connecting recruitment, employee management, task, attendance and payroll workflows.
- **SafeTrack** — A mobile app for parents bringing real-time location tracking, school schedules, and event notifications into one place.

## Tech Stack

- Next.js
- TypeScript / React
- Tailwind CSS
- Figma

The site itself was built with the help of AI-assisted coding tools such as Cursor and Antigravity, while the design work and case studies featured within it are my own.
